# omni-assistant-hub
Welcome to the Omni Assistant Hub
This repository features two decoupled, high-performance Python-based assistant frameworks:

Nova (Local Voice Engine): A completely local, hands-free personal voice assistant built using dynamic microphone ambient noise threshold calibration and fast, zero-latency system text-to-speech synthesis.

Gemini Grounded Chat Node: A stateful, cloud-connected conversational engine driven by the official Google GenAI SDK. It features infinite session context memory handling combined with real-time web RAG (Retrieval-Augmented Generation) utilizing the live Google Search grounding tool layer to answer time-sensitive queries instantly.
